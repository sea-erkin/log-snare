LogSnare is an intentionally vulnerable application which shows how IDOR vulnerabilties can be abused and exploited. Your goal is to go from being a basic user of the boring cyber-security company "LogSnare" to an exciting administrator of the prestigious "Acme Corporation".

Start by clicking on the Dashboard, where hints and instructions will be provided.

Once you're all done, be sure to enable validation and see how all of your malicious actions would've been logged along the way. This is where the real lesson is learned. You can see how web applications should validate, and log IDOR validation requests.
